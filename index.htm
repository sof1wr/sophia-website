About the Method

To find the solutions, these are fundamental ideas:

  1. Add multiples of one equation to the others in order to eliminate a variable.
  2. Continue this process until only one variable is left.
  3. Once this final variable is determined, its value is substituted back into the other equations in order to evaluate the remaining unknowns.


To calculate the inverse matrix, follow these steps:>

  1. Set the matrix (must be square) and append the identity matrix of the same dimension to it.
  2. Reduce the left matrix to row echelon form using elementary row operations for the whole matrix (including the right one).
  3. As a result, you will get the inverse calculated on the right.
  4. If the determinant of the main matrix is zero, the inverse doesn't exist.
